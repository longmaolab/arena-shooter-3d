# How to Play — Arena Shooter 3D

> 📘 English version first. 中文版在下半部分 ([跳到中文](#怎么玩-arena-shooter-3d-中文)).

A one-page guide for players. No setup, no installs — just open a link in your browser.

---

## 🎮 Where to play

Open this link in any browser:

### **https://game.boobank.com/arena-shooter/**

Works on:
- 💻 Desktop / laptop — **Chrome, Edge, Firefox, Safari**
- 📱 Phone / tablet — **iOS Safari, Android Chrome**

> First load takes 10–30 seconds (it downloads the game, ~37 MB).
> After that it stays cached, so opening again is fast.

---

## 💻 Desktop (PC / Mac) controls

| What you want to do | Key |
|---|---|
| Move | **W A S D** |
| Look around | **Mouse** (move) |
| Jump | **Space** |
| Shoot | **Left click** |
| Reload | **R** |
| Switch weapon | **1 / 2 / 3** (pistol / SMG / shotgun) |
| Pause / release mouse | **Esc** |

### First time on desktop

1. Open https://game.boobank.com/arena-shooter/
2. Wait for the **▶ PLAY** button to appear
3. (Optional) Click **<** or **>** to pick a different character (18 options)
4. Click **▶ PLAY**
5. **Click anywhere on the game** — this locks the mouse pointer to the game
6. You're in. Move with WASD, look with mouse, click to shoot. 🎯

> If you want to play alone first (no internet needed), click **🤖 PLAY vs BOTS** instead.

---

## 📱 Mobile (phone / tablet browser) controls

| What you want to do | How |
|---|---|
| Move | **Left thumb stick** (bottom-left of screen) |
| Look around | **Drag the right half of the screen** |
| Jump | **Blue JUMP button** |
| Shoot | **Red FIRE button** |
| Reload | **Yellow RELOAD button** |
| Switch weapon | Tap **PIS** / **SMG** / **SHG** chips |

### First time on mobile

1. Open https://game.boobank.com/arena-shooter/ in **Safari (iOS)** or **Chrome (Android)**
2. **Turn your phone sideways** (landscape) — the game is designed for landscape
3. Wait for the game to load
4. Tap **▶ PLAY**
5. The control buttons appear on screen. Left thumb to walk, right half to look around.

### Mobile tips

- **Tap "Add to Home Screen"** in the browser menu — game opens fullscreen like a real app
- **iOS Safari**: tap **Share button** → **Add to Home Screen**
- **Android Chrome**: tap **⋮ menu** → **Add to home screen**
- After that, tap the icon on your home screen → fullscreen mode, no browser bar

---

## 🎯 Game rules (quick)

- **Goal**: First player to **10 kills** wins the round
- **Score**: 1 kill = 1 point. Headshots do double damage (red `HEAD! -50` floats up)
- **When you die**: black screen for 2.5 seconds → respawn far from enemies → 1.5 s invincibility (you blink)
- **After a round**: 5-second countdown → next round auto-starts, scores reset
- **Pickups** scattered on the map:
  - ❤️ **Health pack** (red) — restores HP
  - 📦 **Ammo crate** (yellow) — refills ammo
  - 🟦 **Jump pad** (blue) — boosts you high into the air

---

## 🔫 Three weapons (1 / 2 / 3 to switch)

| Key | Weapon | Best for |
|---|---|---|
| **1** | **Pistol** (PIS) | Long range, precise, slow fire |
| **2** | **SMG** (SMG) | All-round, fastest fire, default |
| **3** | **Shotgun** (SHG) | Close range, huge damage in face |

Each weapon has its own ammo. Reload with **R** (desktop) or **RELOAD** button (mobile).

---

## 🆘 If something goes wrong

| Problem | Try this |
|---|---|
| Page won't load / blank screen | **Hard refresh**: Cmd+Shift+R (Mac) / Ctrl+Shift+R (PC) / pull-down-to-refresh (mobile) |
| "Connection failed" pop-up | Refresh once. If it keeps failing, ask Dad. |
| Game looks slow / laggy | Close other browser tabs. On phone, restart browser. |
| Can't switch weapon | Press 1 / 2 / 3 (desktop) or tap chip (mobile). If still stuck, refresh. |
| Mouse pointer floats around (desktop) | Click on the game once to lock the pointer |
| Can't see joystick (mobile) | Rotate phone to landscape; if still missing, refresh |

For everything else, ask Dad or check the [longer Kids Guide](KIDS_GUIDE.md).

---

## 🧒 Tips for younger kids

- **Practice vs bots first** — pick **🤖 PLAY vs BOTS** → 1 bot → easy mode
- **Use the SMG** (key **2**) when starting out — it's the most forgiving
- **Watch your HP** (top-left) — when it's low, look for ❤️ health packs
- **Press Space** before jumping over things — don't run into walls
- **Use jump pads** (the glowing blue squares) to surprise enemies from above
- **If someone keeps killing you** — switch character (different look) and try a different weapon

# 怎么玩 Arena Shooter 3D (中文)

一页纸玩家指南。**不用装,不用配,浏览器打开就玩。**

---

## 🎮 在哪里玩

任意浏览器打开:

### **https://game.boobank.com/arena-shooter/**

支持设备:
- 💻 电脑(Windows / Mac) —— **Chrome / Edge / Firefox / Safari** 都行
- 📱 手机 / 平板 —— **iPhone 用 Safari,安卓用 Chrome**

> 第一次加载需要 10-30 秒(下载游戏,约 37MB),
> 之后浏览器会缓存,再打开就很快。

---

## 💻 电脑(PC / Mac)操作

| 你想做什么 | 按哪个键 |
|---|---|
| 移动 | **W A S D** |
| 看四周 | **鼠标**(滑动) |
| 跳跃 | **空格** |
| 射击 | **鼠标左键** |
| 换弹 | **R** |
| 切换武器 | **1 / 2 / 3**(手枪 / 冲锋枪 / 霰弹枪) |
| 暂停 / 释放鼠标 | **Esc** |

### 电脑第一次玩

1. 打开 https://game.boobank.com/arena-shooter/
2. 等 **▶ PLAY** 按钮出现
3. (可选)点 **<** 或 **>** 切换角色(18 个方块小人随便挑)
4. 点 **▶ PLAY**
5. **点一下游戏画面** —— 鼠标会被"锁住",这样你转头才顺
6. 进游戏了。WASD 走路,鼠标看四周,左键打枪。🎯

> 想先一个人练习(不用联网),点 **🤖 PLAY vs BOTS** 就行。

---

## 📱 手机 / 平板 浏览器操作

| 你想做什么 | 怎么做 |
|---|---|
| 移动 | **左下角虚拟摇杆** |
| 看四周 | **手指滑动屏幕右半边** |
| 跳跃 | 蓝色 **JUMP** 按钮 |
| 射击 | 红色 **FIRE** 按钮 |
| 换弹 | 黄色 **RELOAD** 按钮 |
| 切换武器 | 点 **PIS** / **SMG** / **SHG** 标签 |

### 手机第一次玩

1. 浏览器(iPhone 用 **Safari**, 安卓用 **Chrome**)打开 https://game.boobank.com/arena-shooter/
2. **手机横过来** —— 游戏是横屏设计
3. 等游戏加载完
4. 点 **▶ PLAY**
5. 屏幕上出现操作按钮。左手拇指走路,右半屏滑动看四周。

### 手机小贴士

- **加到主屏幕** —— 之后从图标点进去,**全屏没浏览器栏**,像 App 一样
- **iOS Safari**:点 **分享按钮** → **添加到主屏幕**
- **Android Chrome**:点 **⋮ 菜单** → **添加到主屏幕**
- 加完之后,从主屏图标进入 = 真·全屏游戏

## 🎯 游戏规则(速览)

- **目标**:第一个达到 **10 杀** 的人赢一局
- **得分**:杀 1 个 = 1 分。**爆头双倍伤害**(头上飘红字 `HEAD! -50`)
- **死了之后**:黑屏 2.5 秒 → 在远离敌人的复活点重生 → 1.5 秒无敌(角色闪烁)
- **一局结束后**:屏幕中间 5 秒倒计时 → 自动开下一局,分数清零
- **地图道具**:
  - ❤️ **红色血包** —— 回血
  - 📦 **黄色弹药箱** —— 补子弹
  - 🟦 **蓝色跳台** —— 弹很高,可以越障/突袭

## 🔫 三把武器(按 1 / 2 / 3 切换)

| 数字键 | 武器 | 适合场景 |
|---|---|---|
| **1** | **手枪**(PIS) | 远距离,精准,射速慢 |
| **2** | **冲锋枪**(SMG) | 万金油,射速最快,默认 |
| **3** | **霰弹枪**(SHG) | 近战之王,贴脸秒杀 |

每把枪有独立弹夹。**R** 换弹(电脑)或点 **RELOAD** 按钮(手机)。

## 🆘 出问题了怎么办

| 问题 | 试试这个 |
|---|---|
| 页面打不开 / 全白 | **硬刷新**:Mac 上 Cmd+Shift+R / Win 上 Ctrl+Shift+R / 手机下拉刷新 |
| 弹 "Connection failed" | 刷新一下。还是不行就问爸爸。 |
| 游戏很卡 | 关掉浏览器别的标签页。手机重启浏览器。 |
| 换不了武器 | 电脑按 1/2/3,手机点底部标签。还是不行就刷新。 |
| 鼠标飘来飘去(电脑) | 点一下游戏画面,鼠标会被锁定 |

其它问题问爸爸,或者看更详细的 [Kids Guide](KIDS_GUIDE.md)。

## 🧒 给小朋友的小建议

- **先跟 Bot 练手** —— 选 **🤖 PLAY vs BOTS** → 1 个 Bot → 简单模式
- **新手用冲锋枪**(按 **2**)—— 子弹多,容错高
- **盯着左上角的 HP** —— 血少了赶紧找红色血包
- **跳跃要按空格** —— 别撞墙
- **跳台**(蓝色发光方块)可以让你跳很高 —— 从空中突袭敌人很爽
- **被打惨了** —— 换个角色(换换运气),换把枪试试别的打法
